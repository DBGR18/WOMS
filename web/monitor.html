Grafana 儀表板尚未啟動

監控儀表板需要 Grafana 服務。請確認 docker compose up 已包含 prometheus、grafana 與 gthulhu 服務。

docker compose up -d postgres redis kafka api scheduler-worker web gthulhu prometheus grafana

Grafana 預設在 localhost:3000 運行，已設定為匿名唯讀存取。 當 Grafana 就緒後，重新整理此頁面即可載入儀表板。
